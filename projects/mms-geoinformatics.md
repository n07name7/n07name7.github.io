# Geoinformatics Multimedia System

## Overview
Educational web platform showcasing GIS capabilities, spatial analysis examples, and interactive map galleries. Created as academic project at VÅ B-TUO.

## Technologies
- HTML5 / CSS3
- JavaScript
- Lightbox for image galleries
- Responsive design
- Interactive map embeddings

## Features
- **Multi-page navigation** - Organized content structure
- **Interactive gallery** - GIS maps and visualizations with zoom functionality
- **Video integration** - Educational content about GIS
- **Audio elements** - Enhanced user experience
- **Localized content** - Czech language for local audience

## Purpose
Demonstrates practical application of geoinformatics through visual examples and interactive elements. Covers fundamental GIS concepts and real-world spatial data analysis.
