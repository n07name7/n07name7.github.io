# Population Change Map

## Overview
Interactive web map visualizing demographic changes across regions. Built using QGIS and exported as web application with QGIS2Web plugin.

## Technologies
- QGIS for spatial analysis and cartography
- QGIS2Web for web export
- OpenLayers for interactive visualization
- Custom styling and layer controls

## Features
- Interactive pan and zoom
- Layer switching
- Pop-up information on click
- Responsive design
- Thematic visualization of population dynamics
